# 🌤️💵 BOTweather

Многофункциональный Telegram-бот для получения актуальной информации о погоде и курсах валют.

## 📋 О проекте
Бот помогает быстро узнавать прогноз погоды в любом городе мира и отслеживать курсы основных валют прямо в Telegram.

## 🚀 Функционал

### Погода:
- 🌡 Текущая температура
- 🤔 Ощущаемая температура
- ☁️ Описание погоды (ясно, облачно, дождь и т.д.)
- 💧 Влажность
- 💨 Скорость ветра

### Курсы валют:
- 💵 USD (Доллар США)
- 💶 EUR (Евро)
- 💷 GBP (Фунт стерлингов)
- 🇨🇭 CHF (Швейцарский франк)
- 🇨🇳 CNY (Китайский юань)

## 🛠 Технологии
- Python 3.x
- python-telegram-bot
- Requests (для API запросов)
- OpenWeatherMap API
- ExchangeRate-API / Центробанк РФ

## ⚙️ Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/siroptull-tech/BOTweather.git
cd BOTweather
Установите зависимости:

bash
pip install -r requirements.txt
Получите API ключи:

Для погоды: OpenWeatherMap (бесплатная регистрация)

Для валют: ExchangeRate-API или используйте бесплатный API Центробанка

Запустите бота:

bash
python bot.py
